Administrator

Datentypen, Attribute oder Objekttypen mit GET abrufen und mit POST hinzufügen.

[GET|POST] /admin/atyp
[GET|POST] /admin/attr
[GET|POST] /admin/otyp

Die gespeicherten Formulare mit GET abrufen und und mit POST ein neues Formular hinzufügen. Beim GET optional über den Query-Parameter "edit" ein Formular zum Umbenennen ergänzen. Ein vorhandenes Formular über dessen ID per POST umbenennen.

[GET|POST] /admin/form
[GET] /admin/form?edit={formId}
[POST] /admin/form/{formId}

Benutzer

Gespeicherte Objekte mit GET abrufen und mit POST ein neues Objekt hinzufügen. Bei der Neuanlage wird der Objekttyp erwartet und optional die Bezeichnung (das Attribut 0).

[GET|POST] /admin/obj
GET /admin/obj?query={search}

Ein Objekt und seine Attribute abrufen. Die Auswahl der Attribute kann über das Formular eingeschränkt werden.

GET /admin/obj/{objId}/attr
GET /admin/obj/{objId}/attr?form={formId}
